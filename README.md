# Project: Freeze Me!

## Prerequisites
- Node.js muss installiert sein
- Im Ordner ```frontend/``` ```npm install``` ausgeführt
- Im Ordner ```backend/``` ```pip install -r .\requirements.txt``` ausgeführt

## Programm starten
1. ```cd frontend```
2. ```npm run serve```
3. ```cd ..```
4. ```cd backend```
5. ```uvicorn app.main:app --reload```
